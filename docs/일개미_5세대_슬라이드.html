2분기: 5세대 출시 & 서비스 안정화

4월 출시, 6월까지 실 서비스 안정화
4월 — 5세대 출시
  • SW 출시 (4/13), HW 납품 (4/9), 오픈형 모델 (4월 말)
  • QC 툴, 태스크 시퀀스 안정화, 무선 충전 도킹 안정화
  • Gaemi Triage Acuity System + LED 점등 통합

  • 로보이츠 쿠폰 시스템 출시
5월 — 서비스 확장
  • EV 연동 서버화 (현대), 공동현관/자동문 연동
  • 주행로 인식 모델 정기 학습 도입
6월 — 안정화 & 기반 구축
  • 하드웨어 판매 — 오픈 코드, 매뉴얼 제공, HW 커스텀 지원
  • 수거 서비스 런칭 (엠밸리4단지, 덤프4 + 배달2)
  • 호출 → 집앞 이동 → 적재 → 쓰레기장 덤프 → 복귀
  • Gaemi Admin 셀프등록 / Gaemi POS
  • P2P (GAEMI go) UX 설계 · MVP 기획
